Autive Embed Examples

These examples demonstrate how to embed an Autive chat agent on your website.

Where to find this:
Agent ID: Open your agent in Autive, click the three dots menu (top right), and select "Copy Agent ID".
Slug: The slug is shown in your agent's public URL after publishing (e.g., autive.com/chat/my-agent-slug).

Expanded Mode

The chat interface fills 100% of its parent container. Ideal for dedicated chat pages or sections.

data-position="expanded"

Widget Mode

A floating support button in the bottom-right corner that opens a popup chat window when clicked.

data-position="widget"
